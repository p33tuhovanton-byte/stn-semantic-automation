# stn-semantic-automation

### STN: Проект "Наименование" (Семантический автоматизм)

Мультиязычная экосистема (**Go, Rust, Python, R**), реализующая принцип полного разделения имени и сути объектов.

### 🧠 Фундаментальное правило

Наименование (например, слово `транзакция`) само по себе абсолютно **пусто и не имеет контекста**.  
Вся бизнес-логика, правила валидации человека (**Кто делает запрос**) и целевое назначение (**С какой целью / Зачем**) определяются исключительно внешним **Описанием** в формате международного стандарта **JSON Schema**.

Любое изменение в изолированном файле `schemas/transaction.json` мгновенно меняет поведение всех четырех сервисов одновременно без пересборки их исходного кода.
