# stn-semantic-automation

# STN: Проект "Наименование" (Семантический автоматизм)

Мультиязычная экосистема (Go, Rust, Python), реализующая принцип полного разделения имени и сути.

## 🧠 Фундаментальное правило
Наименование (например, слово `транзакция`) само по себе абсолютно **пусто и не имеет контекста**. 
Вся бизнес-логика, правила валидации человека (**Кто**) и целевое назначение запроса (**С какой целью / Зачем**) определяются исключительно внешним **Описанием** в формате международного стандарта **JSON Schema**.

 Любое изменение в файле `schemas/transaction.json` мгновенно меняет поведение всех трех сервисов (Go, Rust, Python) без пересборки их исходного кода.

## 🚀 Порты запуска сервисов:
* **Python (FastAPI + Swagger):** `http://127.0.0`
* **Go (Native HTTP):** `http://127.0.0`
* **Rust (Actix-Web):** `http://127.0.0`
